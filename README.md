> Um projeto nascido da amizade, construído com código e carinho para ajudar o casal Heron e Malu a iniciarem a vida no novo lar.

Este projeto é uma plataforma personalizada de lista de presentes e arrecadação (estilo "vaquinha virtual"). 

## ✨ Como Funciona

- **Navegação Simples:** Os convidados podem filtrar os presentes por cômodos (Sala, Quarto, Cozinha, etc.).
- **Contribuição Flexível:** O convidado escolhe com quanto deseja ajudar na compra de cada item.
- **PIX Integrado:** Geração de QR Code e código "Copia e Cola" dinâmico.
- **Mágica em Tempo Real:** Assim que o pagamento é feito no app do banco, a tela do convidado atualiza sozinha, fechando o QR Code, disparando uma chuva de confetes e agradecendo pelo presente — tudo isso enquanto a barra de progresso do item avança na tela inicial.
